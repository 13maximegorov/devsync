# DevSync

**DevSync** is an instant messaging, audio and video conferencing application for software development teams.

Technology stack:

1. TypeScript
2. React
3. Next.js
4. shadcn/ui
5. Tailwind CSS
6. Zustand
7. Prisma
8. Zod
9. PostgreSQL
10. Socket.IO
11. TanStack Query
12. LiveKit (Golang)
13. ESLint
14. Prettier
15. Docker
16. Caddy

## How to start DevSync

### Development

```zsh
docker-compose up -d
```
